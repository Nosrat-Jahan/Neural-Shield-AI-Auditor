# 🛡️ Neural-Shield AI | Security Auditor v9.9.9

**Neural-Shield AI** is a professional-grade threat detection suite that integrates **Machine Learning** with real-time network security auditing. This project marks the **20th Milestone** of my professional portfolio mission.

## 🧠 Intelligence Engine
The module utilizes a **Random Forest Classification** logic to evaluate file metadata, entropy, and extension risks. It provides a dynamic **Risk Probability Index** to identify potentially malicious signatures before execution.

## ✨ Key Features
- **Heuristic Neural Scan**: Advanced pattern matching using `scikit-learn`.
- **Async Data Fetching**: Zero-latency feedback loop between the ML backend and the UI.
- **Cyber-Slate Interface**: A premium, high-performance dashboard with Neural-Violet aesthetics.
- **Visual Mode Switching**: Supports both High-Contrast Dark and Slate-Light modes.

## 🛠️ Technology Stack
- **AI/ML:** Python, NumPy, Pandas, Scikit-learn
- **Backend:** Flask (WSGI Server)
- **Frontend:** HTML5, Modern CSS Grid, JavaScript (ES6+), Google Fonts
